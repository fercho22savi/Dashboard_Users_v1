# 📄 Generador de Currículum Dinámico (myCV_v2)

Una herramienta web interactiva y moderna para la creación de currículums profesionales, diseñada con un enfoque en la limpieza visual y la facilidad de personalización.

## 🚀 Vista Previa
Puedes ver el proyecto en funcionamiento aquí: [Dashboard Users v1](https://github.io)

## ✨ Características
- **Diseño Responsive:** Optimizado para diferentes tamaños de pantalla gracias a Tailwind CSS.
- **Carga de Imagen:** Permite subir una foto de perfil personalizada directamente desde la interfaz.
- **Secciones Completas:** Incluye apartados para:
  - Resumen Profesional.
  - Historia Laboral y Proyectos Destacados.
  - Educación, Certificaciones y Cursos.
  - Habilidades Técnicas e Idiomas.
  - Referencias Personales.
- **Integración Social:** Enlaces directos a perfiles de GitHub, LinkedIn y Discord.

## 🛠️ Tecnologías Utilizadas
Este proyecto ha sido construido utilizando:
*   **HTML5:** Estructura semántica.
*   **Tailwind CSS:** Para un diseño moderno, rápido y estilizado.
*   **JavaScript (Vanilla):** Manejo de la lógica dinámica y renderizado de datos.
*   **Google Fonts (Roboto & Source Code Pro):** Tipografía clara y profesional.
*   **Font Awesome:** Iconografía intuitiva para secciones de contacto y redes.

## 📁 Estructura del Proyecto
```text
├── index.html          # Estructura principal del dashboard
├── scripts/
│   └── myCV_v2.js      # Lógica de renderizado y manejo de datos
└── styles/
    └── myCV_v2.css     # Estilos personalizados adicionales
```

## ⚙️ Instalación y Uso
1. Clona este repositorio:
   ```bash
   git clone https://github.com
   ```
2. Abre el archivo `index.html` en tu navegador preferido.
