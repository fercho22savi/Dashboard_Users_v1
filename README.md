# 📄 Generador de Currículum Dinámico (myCV_v2)

Una herramienta web interactiva para crear currículums profesionales con un diseño limpio y moderno.

## 🔗 Enlace:
Puedes ver el proyecto funcionando aquí:  
👉 **[https://fercho22savi.github.io/Dashboard_Users_v1/](https://fercho22savi.github.io/Dashboard_Users_v1/)**

---

## ✨ Características
- **Interfaz Intuitiva:** Diseño organizado mediante una barra lateral de contacto y un cuerpo principal para experiencia y educación.
- **Personalización de Imagen:** Función para subir y visualizar tu foto de perfil en tiempo real.
- **Secciones Detalladas:** 
  - Perfil profesional, Experiencia laboral y Proyectos.
  - Educación, Certificaciones y Cursos.
  - Habilidades técnicas y manejo de idiomas.
- **Estilo Moderno:** Uso de tipografías optimizadas (Roboto y Source Code Pro) e iconografía de Font Awesome.

## 🛠️ Tecnologías Utilizadas
*   **HTML5 & JavaScript:** Estructura y lógica dinámica.
*   **Tailwind CSS:** Framework para el diseño responsive y estilizado.
*   **Font Awesome:** Iconos para redes sociales y secciones de contacto.
*   **Google Fonts:** Tipografía profesional para mejor legibilidad.

## 📁 Estructura del Repositorio
*   `index.html`: El núcleo de la aplicación.
*   `scripts/myCV_v2.js`: Maneja la carga de datos y funciones de la interfaz.
*   `styles/myCV_v2.css`: Estilos personalizados que complementan a Tailwind.
